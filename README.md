# LUSH's Blog

这是 LUSH 的个人博客项目，基于 Astro 构建，用来记录技术学习、项目实践、工具折腾和阶段性思考。

## 常用命令

| Command | Action |
| --- | --- |
| `npm install` | 安装依赖 |
| `npm run dev` | 启动本地开发服务器 |
| `npm run build` | 构建生产版本到 `./dist/` |
| `npm run preview` | 本地预览构建结果 |

## 内容结构

- `src/pages/`：站点页面与路由
- `src/components/`：页眉、页脚、日期等复用组件
- `src/content/blog/`：博客文章内容
- `src/styles/global.css`：全局样式

当前样式基于 Astro 官方博客模板，并保留了原模板中对 Bear Blog 样式来源的许可归因。
